## 1. Competitor Overview

- **Company name:** Acuity Scheduling (by Squarespace)
- **Website:** https://acuityscheduling.com

## 7. Flags and Gaps

### Unverified Claims
- **Acuity G2 exact review count:** G2 returned 403 on direct fetch. The rating (4.1/5) is from search snippet. Review count could not be confirmed. Recommend manually checking G2 before publication.
- **Google Calendar / Outlook sync for Acuity:** Not explicitly listed on Acuity's integrations page, but widely mentioned in reviews and third-party articles. Likely available but could not confirm from primary source. The signup page mentions "automatic timezone conversion" which implies calendar integration but does not name Google/Outlook specifically.
- **Acuity "Client Loyalty" feature:** Listed on features page navigation but no detail on what it includes. It is unclear if this is a points-based system or simply refers to repeat booking tools.
- **Acuity POS:** Listed as a feature on the features page navigation but no details available. Could not confirm scope or functionality.
- **SimplyBook.me push notifications:** Mentioned in existing Setmore landing page but not found on the SimplyBook.me features page during this research.
- **SimplyBook.me Zapier integration:** Not listed on the features page. May be available but could not confirm from primary source.

### Stale Data Warnings
- Acuity's pricing page (signup.php) does not display a "last updated" date. Pricing confirmed as current via multiple third-party sources reporting the same figures in 2026.
- The plan name "Emerging" appears in the Squarespace help article but the signup page uses "Starter." The signup page likely reflects the current plan names.

### Contradictions
- **Plan naming:** Squarespace help center references plan names "Emerging, Growing, Powerhouse" while the Acuity signup page uses "Starter, Standard, Premium." The signup page is likely the current naming convention. Earlier search results also referenced "Emerging, Growing, Powerhouse" which may be legacy names. Used the names from the live signup page.
- **Acuity HIPAA plan:** Search results mention "Premium or Powerhouse plan" for HIPAA. The live signup page shows HIPAA BAA on the "Premium" plan. This appears to be the same plan under its new name.

### SimplyBook.me Data Notes
- Pricing on the live website is displayed in EUR (€), not USD ($). The existing Setmore comparison page uses USD figures. The EUR prices are: Free €0, Basic €11.90, Standard €24.90, Premium €49.90. Verify whether the landing page should use EUR or USD and whether there is a USD pricing page.
- The brand voice doc says "12,000 businesses in 100+ countries." This figure was not visible on the pricing page. Recommend confirming this is still the approved figure.
- SimplyBook.me features page content was partially truncated during fetch. Some categories (Statistics & Analytics, Privacy & Security, Notifications, Admin & Client App) were not fully captured. A manual review of the full page is recommended.

## 8. Research Summary

- **Total sourced claims:** ~85
- **Unverified / flagged claims:** 7
- **Key differentiators for SimplyBook.me:**
  - Free plan available (Acuity has none)
  - 40+ payment gateways vs 3 for Acuity
  - HIPAA at Standard plan (~€24.90/mo) vs Acuity Premium ($39-49/mo)
  - SAML SSO (Acuity does not offer)
  - Branded client app (Acuity does not offer)
  - Google Reserve integration (Acuity does not offer)
  - AI Voice Booking (Acuity does not offer)
  - Built-in loyalty points system (Acuity's "Client Loyalty" feature is unclear in scope)
  - Waiting list feature (not listed for Acuity)
  - ISO 27001 certification (Acuity does not list)
- **Key differentiators for Acuity Scheduling:**
  - Unlimited appointments on all plans (SimplyBook.me caps bookings per plan)
  - Built-in invoicing software (SimplyBook.me does not list)
  - Stronger email marketing integration ecosystem (Mailchimp, ActiveCampaign, ConvertKit, etc.)
  - More video conferencing integrations (Zoom, Google Meet, GoToMeeting, JoinMe vs Gruveo)
  - 500+ Zapier integrations on all plans
  - Squarespace website builder integration
  - Packages and memberships available (on Standard+ plan)
- **Recommendation:** Data is sufficient to write a legally compliant comparison page. The 7 flagged items should be manually verified before publication, particularly: (1) Acuity's G2 exact review count, (2) whether Acuity's "Client Loyalty" and "POS" features match SimplyBook.me's equivalents in scope, and (3) SimplyBook.me's Zapier availability. The plan name contradiction (Emerging vs Starter) should be confirmed -- use whatever appears on the live signup page at time of publication.
